a) A company's profit
double
Need decimals, large numbers, and precision

b) A person's weight
int
grams!

c) A book title
String
String of characters

d) Average rainfall in September
int or double
Depending on unit used (e.g. mm)

e) Number of runs scored by a cricketer
int
Will be a whole number from zero into the hundreds

f) A telephone number, including area code
String
Although numeric, this is a string of characters - more likely to concatenate/split (area code/international),
may have special characters, and leading zero may matter

g) A dress size
int
Will always be a whole number between 0 and 50
